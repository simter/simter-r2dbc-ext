# [simter-r2dbc-ext](https://github.com/simter/simter-r2dbc-ext) [[English]]

Simter R2DBC 扩展。

特征：

- 自动根据 maven 的依赖配置构建 `ConnectionFactory` bean
- 支持 `r2dbc-h2`
- 支持 `r2dbc-postgresql`
- 支持 `r2dbc-mssql`
