# [simter-r2dbc-ext](https://github.com/simter/simter-r2dbc-ext) [[中文]]

Simter R2DBC Extension.

Features :

- Auto config `ConnectionFactory` bean by maven dependency
- Support `r2dbc-h2`
- Support `r2dbc-postgresql`
- Support `r2dbc-mssql`
